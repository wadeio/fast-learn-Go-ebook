## About The Project

這是一本我在學習Go的過程所記錄的電子書,目的是透過撰寫文字來釐清Go的基本語法與觀念,而本書的將參考多本Go電子書,希望這個Project可以幫助其他人快速學會Go

Use the `ebook` to get started.

## Getting Started

本書將會列出每一章節的大綱,你可以針對想要看的內容,去點選該章節
